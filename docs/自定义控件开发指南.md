## 完整示例：数据卡片（statCard）

本节以数据卡片为例，逐字段说明一个自定义控件的完整配置。

### 0. 你只需要做两件事

| 步骤 | 动作 | 说明 |
|------|------|------|
| ① | 新建一个 `.ts` 文件 | 例如 `src/myWidgets.ts`，复制下文模板 |
| ② | 在 `src/main.ts` 中 import 它 | 加一行 `import './myWidgets'` |

**只需写注册代码，无需自己开发 API。** `registerCustomWidget` 是从项目中已存在的 `src/config/customWidgetAPI.ts` 导入的，这个文件已经写好了，你只管调用。

### 1. 文件位置

`src/examples/customWidgetDemo.ts`，第 51-93 行（statCard 的 `registerCustomWidget` 调用）。

### 2. 完整代码 + 逐行说明

```ts
// ┌─────────────────────────────────────────────────────────────┐
// │ ↓ 这一行导入的是项目中已有的文件，你不需要自己写 API       │
// │   src/config/customWidgetAPI.ts 已经存在，直接引用即可      │
// └─────────────────────────────────────────────────────────────┘
import { registerCustomWidget } from '@/config/customWidgetAPI'

// ========== 注册一个控件 ==========
registerCustomWidget({

    // ────── ① type：控件的唯一标识 ──────
    // 不能与内置控件重名（button、input、label……）
    // 拖入画布后，控件 DOM id 为 statCard_1、statCard_2……
    // 样式代码中主选择器为 #statCard_1
    type: 'statCard',

    // ────── ② label：库面板显示名 ──────
    label: '数据卡片',

    // ────── ③ icon：库面板图标 ──────
    icon: '\uD83D\uDCCA',   // 📊 的 Unicode 转义，等同于直接写 '📊'

    // ────── ④ jsonConfig：控件的全部配置 ──────
    jsonConfig: {

        // label / icon 与上面保持一致
        label: '数据卡片',
        icon: '\uD83D\uDCCA',

        // ╔═══════════════════════════════════════════════════════╗
        // ║  A. defaultStyleData — 默认样式（不可在属性面板编辑）║
        // ╚═══════════════════════════════════════════════════════╝
        defaultStyleData: {

            // A1. base ─ 控件自身的默认样式
            //     key 用 camelCase 驼峰格式写法，数字类型自动加 px 后缀
            base: {
                width: 180,           // → CSS width: 180px
                height: 100,          // → CSS height: 100px
                backgroundColor: '#ffffff',
                borderColor: '#e0e0e0',
                borderWidth: 1,
                borderStyle: 'solid',
                borderRadius: 8,
                display: 'flex',
                flexDirection: 'column',
                justifyContent: 'center',
                alignItems: 'center',
                gap: 6,               // → CSS gap: 6px
            },

            // A2. sub ─ 子元素的默认样式
            //     key = CSS 类名（必须以 . 开头）
            //     value = camelCase 驼峰格式写法，属性 → 值
            sub: {
                '.card-title': {
                    color: '#999',
                    fontSize: '12px',     // 字符串不会自动加 px，直接原样输出
                },
                '.card-value': {
                    color: '#333',
                    fontSize: '24px',
                    fontWeight: '700',
                },
                '.card-subtitle': {
                    color: '#52c41a',
                    fontSize: '11px',
                },
            },

            // A3. pseudo ─ 控件自身的伪类样式（:hover 等）
            pseudo: {},
        },

        // ╔═══════════════════════════════════════════════════════╗
        // ║  B. mappedProps — 可编辑的子元素样式               ║
        // ╚═══════════════════════════════════════════════════════╝
        // 每条 mappedProps 对应一个子选择器。
        // base 中的默认值会出现在属性面板的"样式编辑"标签页中，
        // 用户可在样式编辑器（CSS 源码）中修改，修改会实时反映到
        // 设计区和预览。
        mappedProps: [
            {
                // 选择器：与 htmlTemplate 中的 class 名一致
                selector: '.card-title',
                // 默认样式值（会被样式编辑器 / CSS 编辑器修改覆盖）
                base: {
                    color: '#999',
                    fontSize: '12px',
                },
                // 该子元素的伪类样式
                pseudo: {},
            },
            {
                selector: '.card-value',
                base: {
                    color: '#333',
                    fontSize: '24px',
                    fontWeight: '700',
                },
                pseudo: {},
            },
            {
                selector: '.card-subtitle',
                base: {
                    color: '#52c41a',
                    fontSize: '11px',
                },
                pseudo: {},
            },
        ],

        // ╔═══════════════════════════════════════════════════════╗
        // ║  C. structuralCSS — 不可编辑的结构样式             ║
        // ╚═══════════════════════════════════════════════════════╝
        // key = CSS 类名，value = CSS 声明数组。
        // 这些属性在属性面板中不可见、不可编辑，始终生效。
        // 适合定义布局属性（display、text-align、position 等）。
        structuralCSS: {
            '.card-title': [
                'display: block',
                'text-align: center',
            ],
            '.card-value': [
                'display: block',
                'text-align: center',
            ],
            '.card-subtitle': [
                'display: block',
                'text-align: center',
                'margin-top: 2px',
            ],
        },

        // ╔═══════════════════════════════════════════════════════╗
        // ║  D. computedProps — 动态计算属性                    ║
        // ╚═══════════════════════════════════════════════════════╝
        // 支持模板变量自动计算，如 {{halfH}} = 控件高度的一半。
        // 数据卡片不需要，设为空数组。
        computedProps: [],

        // ╔═══════════════════════════════════════════════════════╗
        // ║  E. defaultProps — 控件业务属性                     ║
        // ╚═══════════════════════════════════════════════════════╝
        // 这里的每个属性会成为控件的直接属性（widget.title 等）。
        // 模板中 {{prop}} 从这里取值。
        // ⚠️ visible: true 必须设置，否则预览导出不显示！
        defaultProps: {
            title: '总用户数',       // → {{title}} 模板变量
            value: '12,846',        // → {{value}} 模板变量
            subtitle: '较昨日 +12%',// → {{subtitle}} 模板变量
            chartType: ['折线图', '柱状图', '饼图', '面积图'],  // 数组 → 下拉框
            accentColor: '#1890ff', // 颜色字符串 → 颜色选择器
            showSubtitle: true,     // 布尔 → 复选框
            visible: true,          // ⚠️ 必设
            disabled: false,
        },
    },

    // ────── ⑤ htmlTemplate：HTML 模板 ──────
    // class 名必须与 structuralCSS / mappedProps / defaultStyleData.sub 中的
    // selector 完全一致（去掉前面的 . 之后）
    // {{prop}} 会被替换为 defaultProps 或属性面板中用户修改的值
    htmlTemplate:
        '<div class="card-title">{{title}}</div>'      +
        '<div class="card-value" style="color:{{accentColor}}">{{value}}</div>' +
        '<div class="card-subtitle" data-visible="{{showSubtitle}}">{{subtitle}} {{chartType}}</div>',

    // ────── ⑥ cssString：额外 CSS ──────
    // 可选。注入到设计区/导出/预览页面的 <style> 中。
    // 适用于 @keyframes、复杂选择器、复选框联动隐藏等场景。
    cssString:
        '/* 复选框联动：showSubtitle=false 时隐藏副标题 */\n' +
        '.card-subtitle[data-visible="false"] { display: none !important; }',
})
```

### 3. 拖入画布后能做什么

#### 属性面板

选中控件后，属性面板新增「控件属性」区域，自动根据 `defaultProps` 中的字段类型渲染表单：

```
┌──────────────┐
│ 控件属性      │
├──────────────┤
│ title        [ 总用户数    ]  ← 文本输入框
│ value        [ 12,846     ]  ← 文本输入框
│ subtitle     [ 较昨日+12% ]  ← 文本输入框
│ chartType    [ 折线图 ▼   ]  ← 下拉框（数组值）
│ accentColor  [ ■ #1890ff  ]  ← 颜色选择器（#开头的颜色值）
└──────────────┘
```

- 类型判断：字符串 → `<input type="text">`，数字 → `<input type="number">`，布尔 → `<input type="checkbox">`，数组 → `<select>` 下拉框，`#` 开头的颜色值（如 `#1890ff`）→ `<input type="color">` 颜色选择器
- `visible` 和 `disabled` 在「通用属性」区域显示，不在此处

#### 数据绑定链路（修改一个值经历了什么）

以修改 `title` 为例，完整链路如下：

```
┌─── registerCustomWidget 定义阶段 ─────────────────────────────────────────┐
│                                                                          │
│  defaultProps: {                                                         │
│    title: '总用户数',    ← ① 定义 key = "title"，默认值 = "总用户数"      │
│  }                                                                       │
│                                                                          │
│  htmlTemplate:                                                           │
│    '<div class="card-title">{{title}}</div>'    ← ② 模板中 {{title}}      │
│                                                    引用同一个 key         │
│                                                                          │
├─── 控件创建时（拖入画布） ────────────────────────────────────────────────┤
│                                                                          │
│  widget.title = '总用户数'     ← ③ defaultProps 的值复制到控件对象        │
│                                                                          │
├─── 属性面板渲染 ──────────────────────────────────────────────────────────┤
│                                                                          │
│  遍历 defaultProps，跳过 visible/disabled:                                │
│    key = "chartType", value = ['折线图','柱状图','饼图']                    │
│    Array.isArray(value) → 渲染为 <select> 下拉框                           │
│    key = "accentColor", value = '#1890ff'                                 │
│    /^#[0-9a-fA-F]{3,8}$/.test(value) → 渲染为 <input type="color">        │
│    key = "title", value = "总用户数"                                       │
│    typeof value === 'string' → 渲染为 <input type="text">                 │
│                                                                          │
├─── 用户在属性面板修改 title = "新标题" ───────────────────────────────────┤
│                                                                          │
│  store.updateWidget(widget.id, { title: "新标题" })                      │
│    → widget.title = "新标题"           ← ④ 更新控件对象                    │
│                                                                          │
├─── 模板重新渲染（设计区 + 预览） ─────────────────────────────────────────┤
│                                                                          │
│   把 '{{title}}' 替换为 widget.title                                     │
│    → <div class="card-title">新标题</div>  ← ⑤ 实时生效                   │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

**核心规则**：`defaultProps` 的 key 和 `htmlTemplate` 的 `{{key}}` **必须同名**。

创建控件时 `defaultProps.title` 被复制成 `widget.title`（同一回事），属性面板读的是 `widget.title`，模板替换的也是 `widget.title`。只要一个 `"title"` 字符串串起了全部链路。

| 位置 | 写法 | 作用 |
|------|------|------|
| `defaultProps.title` | `title: '总用户数'` | 定义属性默认值 → 创建时变为 `widget.title` |
| `htmlTemplate` | `{{title}}` | 替换为 `widget.title` 的值 |
| 属性面板 | 读取 `widget.title` | 渲染表单字段，用户可修改 |
| 预览导出 | 替换 `{{title}}` 为 `widget.title` | 实时反映修改后的值 |

修改任一值后，`{{prop}}` 模板变量自动更新，设计区和预览同步变化。

#### 样式编辑器

打开「样式编辑」标签页，可以看到控件当前的 CSS 源码：

```css
/* 数据卡片_1 - .card-title */
#statCard_1 .card-title {
  display: block;        /* ← 来自 structuralCSS */
  text-align: center;    /* ← 来自 structuralCSS */
  color: #999;           /* ← 来自 mappedProps */
  font-size: 12px;       /* ← 来自 mappedProps */
}
```

- 来自 `structuralCSS` 的属性（如 `display`）由系统自动维护，修改无效
- 来自 `mappedProps` 的属性（如 `color`）可以修改，改后设计区和预览同步生效

#### 预览

点击「👁️ 预览」按钮，数据卡片以实际 HTML 渲染：

```
┌──────────────────┐
│    总用户数       │  ← 灰色 12px，居中
│                                                 │
│   12,846         │  ← 黑色 24px 粗体，居中
│                                                 │
│  较昨日 +12%     │  ← 绿色 11px，居中
└──────────────────┘
```

## 完整示例：评分控件（starRating）

同一文件 `src/examples/customWidgetDemo.ts`，第 13-46 行。

### 与数据卡片不同之处

| 维度 | 数据卡片 | 评分控件 |
|------|---------|---------|
| **defaultProps 类型** | 全是字符串 | `maxStars: 5`（数字）、`value: 3`（数字） |
| **属性面板表单** | 3 个文本输入框 | 2 个数字输入框 |
| **模板变量** | `{{title}}` `{{value}}` `{{subtitle}}` | `{{value}}/{{maxStars}}` |

数字类型的 `defaultProps` 会在属性面板渲染为 `<input type="number">`。

### 完整代码

```ts
// 注册 API 来自项目已有文件 src/config/customWidgetAPI.ts
import { registerCustomWidget } from '@/config/customWidgetAPI'

registerCustomWidget({
    type: 'starRating',
    label: '评分控件',
    icon: '\u2B50',           // ⭐

    jsonConfig: {
        label: '评分控件',
        icon: '\u2B50',

        defaultStyleData: {
            base: {
                width: 200,
                height: 36,
                backgroundColor: 'transparent',
                color: '#333',
                fontSize: 14,
            },
            sub: {
                '.star-icon':  { color: '#ffa500', fontSize: '20px' },
                '.star-count': { color: '#999', fontSize: '14px', marginLeft: '6px' },
            },
            pseudo: {},
        },

        mappedProps: [
            {
                selector: '.star-icon',
                base: { color: '#ffa500', fontSize: '20px' },
                pseudo: {},
            },
            {
                selector: '.star-count',
                base: { color: '#999', fontSize: '14px' },
                pseudo: {},
            },
        ],

        structuralCSS: {
            '.star-icon':  [ 'display: inline-flex', 'align-items: center', 'gap: 2px' ],
            '.star-count': [ 'display: inline-block', 'vertical-align: middle' ],
        },

        computedProps: [],

        defaultProps: {
            maxStars: 5,        // 数字类型 → 属性面板渲染为 <input type="number">
            value: 3,           // 数字类型
            visible: true,
            disabled: false,
        },
    },

    htmlTemplate:
        '<div class="star-icon">★★★★★</div>' +
        '<div class="star-count">{{value}}/{{maxStars}}</div>',

    cssString: '',
})
```

### 拖入画布后的效果

```
★★★☆☆  3/5
```

属性面板：

```
┌──────────────┐
│ 控件属性      │
├──────────────┤
│ maxStars  [ 5 ]  ← 数字输入框
│ value     [ 3 ]  ← 数字输入框
└──────────────┘
```

## 注册 ↔ 生效 全流程总结

```
① 创建注册文件（如 src/ 下的 myWidgets.ts）
   └─ 从 src/config/customWidgetAPI.ts 导入 registerCustomWidget（项目已有）
   └─ 调用 registerCustomWidget({ type, label, icon, jsonConfig, htmlTemplate })

② 在 src/main.ts 中 import 该文件
   └─ import './myWidgets'

③ 刷新页面（Ctrl+F5）
   └─ 左侧库面板「自定义控件」出现新图标

④ 拖入画布
   └─ 控件显示，属性面板自动渲染 defaultProps 中的字段

⑤ 属性面板修改值
   └─ 模板中 {{prop}} 实时更新

⑥ 样式编辑器修改
   └─ mappedProps 中的属性可编辑，设计区+预览同步

⑦ 预览/导出
   └─ HTML 模板渲染 + structuralCSS + mappedProps + styleData.sub 合并输出
```

**关键文件**：

| 文件 | 谁负责 |
|------|--------|
| `src/config/customWidgetAPI.ts` | 系统已提供，**不需要你写** |
| `src/config/widgetRegistry.ts` | 系统已提供，**不需要你写** |
| `src/components/widgets/CustomWidget.vue` | 系统已提供，**不需要你写** |
| `src/main.ts` | **你只需要加一行 import** |
| `src/你的文件.ts` | **你写注册代码** |

## 四个样式层的关系（一图看懂）

```
生成的 CSS 输出中，同一选择器的 4 层样式会合并成一条规则：

┌─────────────────────────────────────────────────┐
│  #statCard_1 .card-title {                      │
│    display: block;         ← structuralCSS      │
│    text-align: center;     ← structuralCSS      │
│    color: #999;            ← mappedProps 默认值  │
│    font-size: 12px;        ← mappedProps 默认值  │
│  }                                              │
└─────────────────────────────────────────────────┘

         ↑ 用户修改后，color/text-size 变为新值
         ↑ display/text-align 始终不变（不可编辑）
```

| 层 | 来源 | 可编辑？ | 作用 |
|------|------|---------|------|
| `structuralCSS` | jsonConfig | ❌ 不可编辑 | 布局结构（display, text-align, position） |
| `mappedProps` | jsonConfig | ✅ 可编辑 | 视觉属性默认值（color, fontSize, backgroundColor） |
| `defaultStyleData.sub` | jsonConfig | ❌ 预览/导出用 | 与 mappedProps 合并输出 |
| 样式编辑器修改 | 用户操作 | — | 覆盖 mappedProps 默认值 |

## 排查清单

如果控件不生效，按顺序检查：

1. ✅ `src/main.ts` 中有 `import` 你的注册文件
2. ✅ `defaultProps` 中设了 `visible: true`
3. ✅ `htmlTemplate` 中的 `class` 名与 `structuralCSS`、`mappedProps` 的 `selector` 一致
4. ✅ `mappedProps[].selector` 以 `.` 开头
5. ✅ `defaultStyleData.sub` 的 key 以 `.` 开头
6. ✅ 刷新了页面（Ctrl+F5）
